How to Run or Contribute

The primary purpose of development MoilApps is to provide apps that can implement the Moildev-SDK to process fisheye image. This app can use to provide panorama image and Anypoint view.

How to run these apps Follow this step before you use MoilApp,

1. Clone this repository

$ git clone https://github.com/MoilOrg/Moildev-Applications.git

2. Change directory moilapp

$ cd moilapp

To install library have two ways 1. Build MoilApp by using this Command, 2. Install requirements by manually 3. Build MoilApp by using this Command

$ . build.sh

or

$ pip install -r Requirement.txt

3. The running moilapp project

$ python MoildevApps/main.py
